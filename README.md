# Retreat Directory

Working project for a directory focused on executive retreats and remote team offsites.

## Working thesis
Build a curated directory of executive retreat venues and remote-team offsite locations, starting with the U.S. and emphasizing practical planning filters that matter to buyers:
- team size
- airport distance
- lodging capacity
- meeting space
- buyout availability
- setting (mountain, beach, desert, forest, urban)
- budget band
- region

## Core positioning
Not a generic travel list. A planning tool for:
- executive offsites
- leadership retreats
- remote team meetups
- strategy retreats

## Monetization candidates
1. Featured listings
2. Lead generation / inquiry routing
3. Sponsorships
4. Concierge matching
5. Digital planning templates

## Competitive reality
This space has existing players. We need sharper positioning, cleaner filters, and initially narrower scope.

## Initial wedge ideas
- Executive retreat venues in the U.S.
- Remote team retreat venues with lodging + meeting space
- Offsite venues near major airports
- Pacific Northwest / West Coast retreat venues first
