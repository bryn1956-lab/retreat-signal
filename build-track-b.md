# Track B Build Start

Track A reached a sufficient threshold:
- 12 verified venues
- 3 search_verified venues
- 15 total shortlist candidates

## Decision
Begin build using the 12 verified venues only.

## Build principle
Use verified inventory as the publishable base.
Keep search_verified inventory separate until upgraded.

## Immediate build tasks
1. Convert verified venue data into site-ready JSON
2. Create site scaffold
3. Create homepage copy and positioning
4. Create venue card/detail templates
5. Create state landing pages

## Cheapest practical build path
Static site / lightweight app with file-based data.

## Why
- low cost
- SEO-friendly
- easy to maintain
- easy to expand with additional venues later
