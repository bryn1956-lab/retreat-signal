# Analytics

## Current recommendation
Use a lightweight privacy-respecting analytics layer first.

Good options:
- Cloudflare Web Analytics
- Plausible
- Simple Analytics

## Cheapest easiest path
**Cloudflare Web Analytics** if available in the account.

## Why
- simple
- lightweight
- low friction on Cloudflare-hosted pages
